Введение

Что такое Gulp?

Если сильно упростить, то это набор для самостоятельной кастомной сборки вашего проекта. Огромное его преимущество перед другими популярными системами сборки в том, что gulp дает гораздо больше гикости и контроля в решении нестандартных задач.

Главные преимущества Gulp для начинающего пользователя:

  • Автоматическая конвертация шрифтов и запись подключения в CSS файл
  • Работа с ES6 модулями
  • Создание zip-архива с результатом
  • Отправка результата на удаленный FTP-сервер
  • Оптимизация картинок
  • И многое другое!
